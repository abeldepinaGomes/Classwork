meta tag

article tag <article>
Main - defines the main content of a document. Section - defines a section in the document. Article - defines an article in the document.
section tag <section>
a semantic element that enables you to create standalone sections on a web page.
aside tag <aside>
a semantic element introduced to mark up content that is indirectly related to the main article content of a page.
figure tag <<figure>
specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
figure caption tag <figcaption>
defines a caption for a <figure> element.
video embed tag <video>
used to embed video content in a document, such as a movie clip or other video streams.
audio embed tag <audio>
used to embed sound content in a document, such as music or other audio streams
video and audio source tag <source>
used to embed sound content in a document, such as music or other audio streams.
meta tag for charset
an attribute that tells your browser what character encoding to use for this webpage to show the content as desired.
meta tag for keywords
meta tags that you can use to give search engines more information about a page's content
meta tag for viewport
tells the browser that the width of the screen should be considered the "Full Width" of the page.
